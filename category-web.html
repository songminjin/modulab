제작 프로세스
01
카카오 상담 접수
원하는 사이트의 목적, 레퍼런스 URL, 필요 기능을 공유해주세요.
24시간 내 답변
02
기획 & 와이어프레임
페이지 구조와 레이아웃을 먼저 잡고 공유드립니다. 방향 확정 후 디자인 시작.
계약금 50% 선입금
03
개발 & 테스트
디자인·개발 완료 후 미리보기 URL 공유. 수정 의견 반영 후 최종 확정.
수정 3회 포함

납품 & 배포
잔금 입금 후 소스 파일 전달 및 GitHub Pages 배포 지원.
납기 1개월
