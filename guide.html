👤 회원가입 방법
1
소셜 또는 이메일로 가입
상단 "회원가입" 버튼을 클릭하세요. Google, 카카오 소셜 계정으로 간편 가입하거나, 이메일·비밀번호로 직접 가입할 수 있습니다.
만 14세 이상만 가입 가능합니다
2
필수 정보 입력
이름, 닉네임, 이메일, 비밀번호(6~20자), 주소, 전화번호를 입력합니다. 이메일은 디지털 상품 파일 발송에 사용되므로 정확히 입력해 주세요.
3
약관 동의 후 가입 완료
이용약관 및 개인정보 수집·이용 동의(필수)에 체크 후 "등록" 버튼을 누르면 가입이 완료됩니다. 마케팅 수신 동의는 선택 사항입니다.
🔐 로그인 방법

이메일 로그인

가입 시 등록한 이메일과 비밀번호를 입력하여 로그인합니다. 비밀번호를 잊으셨다면 "비밀번호 찾기"를 이용하세요.

소셜 로그인

Google 또는 카카오 계정으로 버튼 클릭 한 번에 로그인할 수 있습니다. 소셜 가입자는 해당 소셜 로그인 버튼을 사용하세요.

보안 주의사항

타인이 사용하는 기기에서는 로그인 후 반드시 로그아웃해 주세요. 계정 도용이 의심되면 즉시 비밀번호를 변경하세요.

계정 관리

마이페이지에서 회원 정보 수정, 주문 내역 확인, 다운로드 이력 조회가 가능합니다.

🛒 구매 절차
1
상품 선택
카테고리별 상품 목록에서 원하는 상품을 클릭하여 상세 페이지를 확인합니다. 상품 설명, 포함 파일 목록, 사용 환경 등을 꼼꼼히 확인하세요.
2
장바구니 또는 즉시 구매
"장바구니 담기"로 여러 상품을 한 번에 구매하거나, "바로 구매"로 즉시 결제 페이지로 이동합니다. 구매는 로그인 후 가능합니다.
3
결제 수단 선택 및 결제
신용카드, 체크카드, 카카오페이, 네이버페이 등으로 결제할 수 있습니다. 결제 정보는 암호화되어 안전하게 처리됩니다.
카드사 무이자 할부 여부는 해당 카드사 정책에 따릅니다
4
결제 완료 & 파일 수신
결제가 완료되면 등록된 이메일로 즉시 파일 다운로드 링크가 발송됩니다. 스팸 메일함도 확인해 주세요. 파일을 받지 못한 경우 고객센터로 문의해 주세요.
즉시 발송 — 영업일과 무관하게 24시간 자동 발송됩니다
💳 결제 수단 안내

신용카드 · 체크카드

국내 모든 카드사 결제 가능. 일부 카드사는 무이자 할부가 적용될 수 있습니다.

카카오페이

카카오페이 앱 또는 카카오톡에 연결된 간편결제로 빠르게 결제할 수 있습니다.

네이버페이

네이버 계정으로 간편하게 결제하며, 네이버페이 포인트 사용도 가능합니다.

계좌이체

실시간 계좌이체로 결제할 수 있습니다. 이체 후 자동으로 결제가 확인됩니다.

📥 파일 다운로드 방법
1
이메일로 다운로드 링크 수신
결제 완료 즉시 가입 이메일로 다운로드 링크가 발송됩니다. 메일이 오지 않았다면 스팸 메일함을 확인하거나, 마이페이지 > 구매 내역에서도 다운로드할 수 있습니다.
2
파일 다운로드 및 압축 해제
링크를 클릭하여 파일을 다운로드하세요. 압축 파일(.zip)의 경우 압축을 해제한 후 사용합니다. 압축 해제 시 한글 파일명이 깨지는 경우 반디집(Bandizip) 또는 알집 사용을 권장합니다.
3
제품별 사용 환경 확인
상품 상세 페이지에 명시된 권장 사용 환경(예: Microsoft Excel 2016 이상, Google Sheets 최신 버전 등)을 반드시 확인하세요. 환경 불일치로 인한 기능 오류는 지원이 제한될 수 있습니다.
🛠️ 상품별 사용 안내

📊 엑셀 프로그램

Microsoft Excel 2016 이상 권장. 매크로·VBA 사용 상품은 콘텐츠 사용 설정을 활성화해야 합니다. (보안 경고 > "콘텐츠 사용" 클릭)

🔧 구글시트 자동화

Google 계정이 필요합니다. 파일을 Google 드라이브에 업로드 후 구글시트로 열어 사용하세요. Apps Script 권한 허용이 필요할 수 있습니다.

🎬 숏폼 템플릿

After Effects, Premiere Pro, CapCut 등 상품 설명에 명시된 프로그램이 필요합니다. 폰트는 별도 설치 후 사용하세요 (폰트 목록 동봉).

🌐 홈페이지 · 프로그램 제작

맞춤 제작 서비스는 별도 협의를 통해 진행됩니다. 카카오 채널 또는 이메일로 요구사항을 전달해 주세요.

결제 완료 후 7일 이내이며, 파일을 다운로드하지 않은 경우 환불이 가능합니다. 또한 판매된 상품이 설명과 현저히 다른 경우, 파일 자체의 심각한 오류가 있는 경우도 환불 대상입니다.
전자상거래법 제17조에 따른 소비자 권리 보장

  • —구매 후 7일이 경과한 경우
  • —맞춤 제작 서비스로 이미 제작이 시작된 경우

🔧 AS (애프터서비스) 안내

모듈랩은 판매한 디지털 상품의 오류 무제한 수정을 지원합니다. 파일 자체의 버그·오류가 발생한 경우, 언제든지 고객센터로 문의해 주세요.

AS 적용 범위

파일 자체의 수식 오류, 매크로 오작동, 기능 미작동 등 상품 자체의 결함에 한해 무상 지원합니다.

AS 미적용 범위

구매자가 파일을 임의 수정하여 발생한 오류, 사용 환경 불일치(구버전 엑셀 등), 기능 추가 요청은 AS 범위에서 제외됩니다.

💬 자주 묻는 질문
Q  결제 후 파일이 이메일로 오지 않아요. ▾
스팸 메일함을 먼저 확인해 주세요. 스팸 처리가 된 경우 발신자(noreply@modulab.co.kr)를 수신 허용 목록에 추가해 주세요. 그래도 없다면 마이페이지 > 구매 내역에서 직접 다운로드하거나, 고객센터(songminjin123@gmail.com)로 주문번호를 첨부하여 문의해 주세요. 당일 재발송해 드립니다.
Q  파일을 몇 번까지 다운로드할 수 있나요? ▾
구매 후 마이페이지에서 언제든지 재다운로드가 가능합니다. 별도의 다운로드 횟수 제한은 없습니다.
Q  구매한 파일을 수정해서 사용해도 되나요? ▾
개인 업무 용도의 수정·활용은 허용됩니다. 단, 수정한 파일을 제3자에게 재배포하거나 판매하는 행위는 저작권법에 따라 엄격히 금지됩니다. 팀 내부 공유는 라이선스 정책에 따라 다를 수 있으므로 상품 상세 페이지를 확인해 주세요.
Q  회사 전체에서 사용하려면 별도 라이선스가 필요한가요? ▾
기본 라이선스는 구매자 1인 사용이 원칙입니다. 팀·기업 단위로 사용하시려면 고객센터로 문의해 주시면 기업용 라이선스 견적을 안내해 드립니다.
Q  엑셀 파일 열 때 "매크로 비활성화" 경고가 떠요. ▾
VBA 매크로를 포함한 상품은 엑셀 보안 설정에 따라 경고 메시지가 표시됩니다. 파일 상단의 노란 보안 경고 바에서 "콘텐츠 사용" 버튼을 클릭하시면 정상 작동합니다. 모듈랩의 파일에는 악성코드가 포함되어 있지 않습니다.
Q  구글시트 파일에서 권한 허용을 요청해요. ▾
Apps Script(Google Sheets 자동화 스크립트)가 포함된 파일은 처음 실행 시 Google 계정 권한 허용을 요청합니다. "고급" > "안전하지 않음(계속)"을 클릭하여 진행해 주세요. 이는 구글의 외부 앱 인증 정책으로 인한 것이며, 모듈랩의 스크립트는 스프레드시트 범위 내에서만 작동합니다.
Q  맞춤 제작은 어떻게 의뢰하나요? ▾
카카오 채널 또는 이메일(songminjin123@gmail.com)로 문의해 주세요. 필요한 기능, 사용 목적, 예산 등을 알려주시면 상세 견적과 일정을 안내해 드립니다. 제작 의뢰 전에 무료 상담도 가능합니다.
Q  회원 탈퇴는 어떻게 하나요? ▾
마이페이지 > 계정 설정 > "회원 탈퇴"에서 진행하실 수 있습니다. 탈퇴 후 구매 이력은 관련 법령에 따라 일정 기간 보관됩니다. 탈퇴 후에는 동일 이메일로 재가입이 30일 후부터 가능합니다.
